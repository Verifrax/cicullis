Enforcement action reference.

This route defines action classes only. No action is executed from this host.

Reference actions

  • Reject — input or state is outside allowed boundary.
  • Defer — insufficient signal for deterministic resolution.
  • Accept — action is within published authority and boundary.
  • Escalate — state requires authority review before any execution.
